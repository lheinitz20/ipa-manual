# Hintergründe

```jsx
<img className={styles.start_background} src="/start_background.png" alt="start_background"></img>
```

Die Hintergründe wurden ebenfalls mithilfe des <b>img-Tags</b> eingefügt. Der Ablauf um einen Hintergrund zu ersetzen ist mit dem der Icons identisch.

<Callout>
    Wie die Grösse oder auch die Position verändert werden kann wird [hier](/docs/adjustments/styles) erklärt.
</Callout>

{
// Evtl. trotzdem auf dieser Seite erläutern

Die Hintergründe wurden mithilfe von verschiedenen Eigenschaften im Stylesheet <b>Animation.module.css</b> platziert. Diese mit TailwindCSS zu herumzuschieben, wäre zu aufwändig/kompliziert gewesen.

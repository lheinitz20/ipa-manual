# Icons

Icons wurden mithilfe eines <b>img-Tags</b> eingebunden.

```jsx
<img src="/icons/photographie_camera.svg" alt="photographie"></img>
```

## Ersetzen
Dadurch sind diese ziemlich leicht zu ersetzen. Das neue Icon wird in das <b>/icons</b> Verzeichnis verschoben. Hat das neue Icon den gleichen Namen, so muss im Code nichts getan werden. 
Hat die Datei allerdings einen anderen Namen so muss beim `src` Attribut der Name der Datei angepasst werden.
